---------------------------------------------------------------------------------
INFO
---------------------------------------------------------------------------------
Searching the shortest path (Dijkstra's algorithm) from the first node to others

---------------------------------------------------------------------------------
COMPILATION
---------------------------------------------------------------------------------
make

---------------------------------------------------------------------------------
USING
---------------------------------------------------------------------------------
./main <input_file.txt> <output_file.txt>

---------------------------------------------------------------------------------
FORMATTING
---------------------------------------------------------------------------------
Input format:
<From node> <To node> <Path weight>

Output format: 
<Final node number> <Shortest path weight> <Previous node>
Shortest path weight = -1   -   path doesn't exist
Previous node = -1   -   previous node doesn't exist
